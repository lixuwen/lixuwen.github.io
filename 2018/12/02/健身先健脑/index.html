健身先健脑

发表于 2018-12-02 | 分类于 Body Building

随着生活水平的提高，人们的健康意识也随之提高，越来越多的人加入到健身的队伍中，每天健身房里的人都是爆满，有真正练的好的，也有“玩”器械的。

健身的好处不计其数，这些好处让许多人对健身非常痴迷，在会籍的引领下办理了健身卡走进了健身房，想要通过健身来改变自己的形态，也想要通过健身来成就自己的人生！

有这个想法着实不错，说明我们已经迈开了第一步，但是我们千万不能认为健身是一件随随便便就能做好的事情，健身是一件需要讲究科学的事情，请初入健身房的人士千万不能低估健身这件事。

健身先健脑，在开始健身之前，我们必须要学习一些基础的健身知识，这样才能让我们更好的起步，才能让我们收获事半功倍的健身效果，更不会让我们刚去健身就导致身体受伤。

今天借此机会把我自己的一些健身经验分享给大家，和大家一起学习一下初入健身房，我们需要掌握的那些基础知识。

热身

如果刚到健身房就去力量区撸铁，那么受伤的概率会非常大，为避免健身受伤，我们一定要在健身前做好全面的热身工作。

  1. 我个人一般都是先在跑步机上快走5分钟（速度6km/h，坡度6%，如果时间允许，我建议是10分钟），同时把肩关节预热一下，低强度的有氧运动可以提高我们的身体温度，告诉我们全身的器官和肌肉，我要开始运动啦。

  2. 跑完步之后再有针对性的做几组小重量器械运动，这是为了告诉我们的目标肌肉，接下来要锻炼你了，请你做好准备💪。

  3. 我每次健身前都会把肩关节热身2分钟，因为肩关节是人体活动范围最大的关节，同时也是比较容易受伤的关节，如果肩关节受伤，那么你只能练腿了！

  4. 如果说除了肩关节还有一个关节要额外热身的话，那么我觉得应该是腕关节，腕关节是很容易被忽略的一个关节，这个关节平时不会刻意去锻炼(如果你平时刻意去锻炼了，那么说明你是健身高手)，如果碗关节受伤，那么你的上半身基本是锻炼不了了。

锻炼

  1. 在热完身之后，真正开始锻炼目标肌肉之前，我都是先来两组中等偏小（也就是比热身时重量稍大一点点）重量的力量锻炼，这样做的目的就是把血液压入目标肌肉中。

  2. 锻炼目标肌肉时采用重量递增的方式进行，一来是给目标肌肉一个准备，二来是激活一下参与的肌肉群；如果一上来就大重量，很难避免不受伤。

  3. 我和我的小伙伴们一起锻炼时，针对目标肌肉采用的方式一般都是做4-5个动作，每个动作做4-5组，每组8-12个，有时会增加一组动作或是多做一个超级组，每组动作做到力竭；当然，你可以采用少动作多组数的方式来锻炼目标肌肉。

  4. 既然是科学健身，那么我们的健身动作一定要是科学的，如果没有认识到这一点，那么就会导致我们的健身效果大大折扣，并且身体受伤的风险加大，甚至会导致受伤，做动作时一定要慢慢来，把动作做到位，让目标肌肉进行充分的顶峰收缩和舒张，这样才能让目标肌肉得到很好的锻炼。

  5. 重量过大或过小都不利于目标肌肉的锻炼，重量过大特别容易导致受伤，同时为了克服比较大的重量，我们会不自然的动作变形和带入其它肌肉一起参与发力，这样其它肌肉代偿过多，不利于目标肌肉的刺激，重量过小对目标肌肉刺激太小，也起不到锻炼的效果，所以，一定要选择合适的重量来锻炼。

拉伸

  1. 锻炼完后一定要进行拉伸，锻炼时我们的目标肌肉一直处于高强度的收缩状态，即使我们锻炼完之后，目标肌肉也会长时间处于紧张状态，所以，我们锻炼完之后一定要把目标肌肉充分拉伸放松一下，这样才有利于目标肌肉的肌纤维增长。

  2. 如果有时间和条件的话可以来一节专业的拉伸课。

饮食

  1. 健身届流行一句话，三分炼，七分吃，虽然没有非常科学的论证这个三七占比，但是，如果健身时不会吃，那么我们可能获得事倍功半的健身效果，甚至有可能导致身体没有足够的营养补充而出现身体体质越来越差的现象。

  2. 健身时，我们要改变我们的饮食，主要宗旨就是高蛋白，低脂肪，少碳水。牛肉、鸡胸、鱼肉、牛奶、豆制品、鸡蛋白，都含有丰富的优质蛋白，同时我们也要补充一些碳水化物，常见的碳水化物有馒头、米饭和五谷杂粮等，蛋白质配合碳水化合物才能给予肌肉最好的修复，蔬菜、水果含有丰富的矿物质、微量元素和一些必要的维生素，帮助我们修复肌肉。

  3. 关于蛋白粉（或是增肌粉）有很多误区，并不是说喝了蛋白粉肌肉围度就会变大，只有经过科学的锻炼和饮食才能促进肌肉的增长，蛋白粉只是一种蛋白含量比较高的补剂而已，喝蛋白粉比吃含蛋白的食物方便了很多。

休息

  1. 健身时，以上都按照科学的方式进行了，那么你离拥有漂亮的肌肉就差最后一步了，那就是充足的休息，高强度锻炼时，我们的身体处于极度活跃状态，全身的器官都在配合着目标肌肉进行必须的供给，这时我们不会感觉到劳累并且会有一定的小兴奋，可想而知，兴奋过后随之带来的就是疲惫，如果感觉到疲惫那么一定要好好休息，休息好了，肌肉才会更好的生长。

  2. 如果前一天没有休息好，那么肯定会影响今天的锻炼，并且容易注意力不集中而导致受伤，与其带着疲惫的身体锻炼不如休息好了再来一次充分的锻炼，所以，一定要休息好了再进行锻炼。

总结

以上都是自己比较推崇的一些建议，篇幅有限，后续会针对一些细节问题进行展开讲解，如果有误，欢迎指正。

希望你拥有一个好身体，早日拥有一个完美的身材。
