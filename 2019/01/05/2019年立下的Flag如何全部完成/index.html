2019年立下的Flag如何全部完成?

发表于 2019-01-05 | 分类于 总结

2019年的第一个星期过的比以往任何一个星期都快

2018年年底为2019年立下的Flag也记得清清楚楚（还没忘记呢），看书、学英语和陪孩子讲故事都在有序的进行着，希望自己能一直坚持进行下去。

现在回头看看当时自己立下的Flag的确有一定的难度，并且都是可量化的目标，既然豪言都说出去了，剩下的就只有对自己下狠手了，况且不对自己狠点怎么能深刻的认识自己呢？

目标已经定好了，那么现在就来讲一下我将如何去把所有立下的Flag一一实现吧。

1、写52篇文章

平均每周写一篇文章这个目标对我来说，我觉得其实一点都不难，多写写文章是自己很早的一个想法，理想是很丰满的，现实却是骨感的，由于自己一直没有坚持下来，所以就给搁置了。

2018年年底，自己写了几篇文章，感觉还不错，所以就把这个目标定在2019年来实现了，实施计划如下：

1、以自己感兴趣的话题为题材

2、提前准备题材、梳理题纲，不能全部放到周末来完成

2、撰写6篇专利

2018年撰写专利的数量为0，原因很简单，就是一个字-懒。其实专利和艺术品一样，都是源于生活而高于生活，多从生活、工作中提取素材和想法，相必会有写不完的专利点。

1、先从工作中找专利点

2、拿不定主意的点子与同事多请教请教

3、把合适的点子写出来，不写总是找不到写专利的感觉

3、看完10本书

虽然每年都是零零散散的看那么几本书，到后来总感觉跟没看似得。原因也很简单，读书笔记没有做好，没有提炼出书本的精华。

1、看过的书一定要做好读书笔记

2、书籍不局限于技术类

4、分享5次

分享可以是部门内的分享，也可以是部门外的分享，所讲的内容不仅是自己熟知的，更是自己深入研究的内容。

分享是最需要时间来准备的，每一次分享都要经过精心的准备和多次试讲，这样的分享才有价值。

1、提前准备分享的素材和PPT

2、机会总是留给有准备的人的

5、学365天英语

这个目标目前来看是比较简单的，因为自己在过去的500多天中一直在坚持学英语，用英语的场合比较少，自己忘记的也比较快，但是不坚持学的话忘的不是更快么？

坚持下去，总会有收获的。

6、5KM跑进20分钟30秒

这个目标没准是最具有挑战性的。

以前从没想过自己会去跑步，觉得自己体重大，跑步会伤膝盖；一次偶然机会，和小伙伴一起在奥森南园跑了5KM，感觉状态还不错，就一直坚持跑了一段时间，觉得5KM的距离根本不具有挑战性，最具挑战性的是缩短完成时间。

其实跑步不仅可以减脂，还可以提高心肺能力，对力量训练还有很大的帮助，何乐而不为呢？

1、循序渐进，一定不能急于求成

2、热身和跑后拉伸很关键

7、跑完1次马拉松

马拉松总长42.195KM，这个距离在骑自行车的时候，都是刚刚开始，而在跑步却是终极挑战。

每次看到别人跑马拉松自己都有想跑的冲动，这次下决心要在19年跑完一次马拉松，此时不跑更待何时？

1、坚持跑步，坚持锻炼，慢慢提高耐力

2、不在乎成绩，关键是在于突破自己内心的障碍（因为没有参加过，内心总会有一点恐惧）

3、报名成功就参加马拉松，不成功就在奥森实现

8、练出6块腹肌

其实每个人都有腹肌，只是有的人明显，有的人不明显而已。

这个没有捷径，只有坚持锻炼腹部肌肉和控制饮食。

2019年应该可以炫腹的。

9、每天陪孩子讲1次故事

想想这个目标真的也不容易，不是自己努力就可以完成的，还需要孩子的配合，更需要365天的坚持

转眼间，孩子就会长大，希望孩子可以在老爸所讲的故事中慢慢长大，慢慢成长。

家长每天上班都特别忙，下班又想好好休息休息，哪有时间陪孩子呢？

时间就像海绵的里水一样，挤挤总会有的。放下手机会发现另一个另一个美好的世界。

1、把陪孩子讲故事这件事情提前，能在早晨讲的就在早晨讲，一定不能拖到晚上下班回家

2、提前准备故事

10、出国旅行1次

旅行是非常美好的事情，不仅可以放松心情，还能体会目的地的风土人情。

2015年在台湾自由行的情景现在还记得清清楚楚，那里有质朴的人民，美丽的风景，物美价廉的美食，想到这里还想再去台湾旅行一次。

2019年的目的地是哪里呢？还没想好呢。

2019年的目标能不能全部实现，2019年12月31日见晓。坚持✊，努力💪。

再见2018，你好2019
在家到底能不能健身?
